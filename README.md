## Deskripsi 
Program ini adalah aplikasi simulasi fisika berbasis Python yang memodelkan gerak osilasi harmonik sederhana pada sistem massa-pegas. Simulasi ini dirancang untuk menunjukkan perbedaan fisis antara penggunaan pegas tunggal, susunan seri, dan susunan paralel secara *real-time*.

### Fitur Utama:
- **Interactive UI**: Mengatur Konstanta Pegas ($k$) dan Massa ($m$) melalui Slider.
- **Dynamic Configuration**: Mendukung 3 mode susunan pegas (Single, Seri, Paralel).
- **Real-time Data Output**: Menampilkan nilai numerik Periode ($T$), Frekuensi ($f$), dan Gaya Hooke ($F$).
- **Visualisasi Grafik**: Grafik gelombang sinus yang menunjukkan posisi benda terhadap waktu ($y$ vs $t$).
